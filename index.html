Box modeling

MARGIN

El 'Margin' es un espacio invisible que hay alrededor de la caja. Alejenado así al resto de elementos de la caja.

PADDING

El relleno es el espacio en blanco alrededor del contenido, este se encuentra entre el borde y el área del contenido.

BORDER

El borde de la caja envuelve el contenido y el de relleno.

OUTLINE

Es similar al border pero la diferencia es que el ouline no ocupa espacio, son dibujados por encima del elemento.

BACKGROUND-COLOR

Es una propiedad de CSS que define el color del fondo de un elemento, puede ser el valor 'color' o 'transparent'

POSITION

Especifica como un elemento es posicionado en el documento. Las propiedades 'top', 'right', 'bottom' y 'left' determinan la ubicación final de los elementos posicionados.

DISPLAY

El display especifica si un elemento es tratado como block or inline element y el diseño usado por sus hijos, como flow layout(Diseño de Flujo), grid(Cuadricula) o flex(Flexible).

WIDTH

El width especifica la anchura del area de contenido de un elemento.

Height

El height especifica la altura de un elemento.
